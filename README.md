Prediction Using Machine Learning
Project Overview
The Big Mart Sales Prediction project is a Machine Learning-based system developed to predict
product sales across various Big Mart outlets. The project analyzes historical retail data and uses
machine learning algorithms to estimate future sales values.
Features
- Data preprocessing and cleaning
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Linear Regression Model
- Sales Prediction
- Performance Evaluation
- Visualization of Results
Technologies Used
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook
Dataset Information
The dataset contains sales-related information for products sold at Big Mart stores including Item
Weight, Item Visibility, Outlet Type, Item MRP, and Item Outlet Sales.
Installation & Setup
1. Clone Repository
2. Install required libraries using requirements.txt
3. Run the project using python main.py
Machine Learning Model
The project uses Linear Regression, a supervised learning algorithm suitable for predicting continuous
numerical values.
Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R2 Score
Future Scope
- Random Forest Regressor
- XGBoost Algorithm
- Real-Time Prediction Dashboard
- Web Application Deployment
Conclusion
The project successfully demonstrates the use of machine learning for retail sales forecasting and
helps improve inventory management and business decision-making.
